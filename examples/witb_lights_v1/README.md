## How To Use

1. Create script entities from `office_witb_lights_scripts.yaml` (or copy/tune inputs for your room).
2. In WITB+ Actions automation, set:
   - `lights_on_script`
   - `lights_off_script`
3. Use `witb_actions_binding_snippet.yaml` as a template for that automation input block.
