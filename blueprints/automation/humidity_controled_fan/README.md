# humidity_controled_fan/

Humidity Controlled Fan blueprint.

## Current version

**v1** — see [`v1/README_humidity_controled_fan_v1.md`](v1/README_humidity_controled_fan_v1.md)

## Summary

Controls a bathroom exhaust fan using a humidity delta sensor. Turns the fan on when the delta exceeds a threshold for a minimum duration, turns it off when the delta drops below a lower threshold, and enforces minimum and maximum runtimes for safety.
