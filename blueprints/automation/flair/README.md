# flair/

Smart Vents - Flair automation blueprint.

## Current version

**v1** — see [`v1/README_flair_v1.md`](v1/README_flair_v1.md)

## Summary

Controls Flair smart vents for a single room based on the WITB+ `occupied_effective` occupancy signal. Sets the vent Active on occupancy and Inactive on vacancy, with reconciliation to correct drift from manual Flair app changes or cloud state resets.
