# zooz_all_light_switch_modified/

Z-Wave Plus Zooz 700/800 Series switch button automation blueprint.

## Current version

**v1** — see [`v1/README_zooz_all_light_switch_modified_v1.md`](v1/README_zooz_all_light_switch_modified_v1.md)

## Summary

Maps physical button presses (1x–5x, held) on Zooz ZEN71/72/76/77 switches and dimmers to configurable actions. Supports optional ON/OFF hook scripts for WITB+ light profile integration.
